# HTML 뷰어 - 치과 3D 모델 뷰어 앱

iOS 아이폰에서 치과 3D 모델 HTML 파일을 쉽게 열고 볼 수 있는 Progressive Web App (PWA)입니다.

## 주요 기능

- **카카오톡/메신저에서 바로 열기** - 파일을 저장할 필요 없이 "공유" 기능으로 바로 열기
- HTML 파일 선택 및 뷰어
- 최근에 열어본 파일 목록 (최대 10개)
- iOS 홈 화면에 앱으로 추가 가능
- 오프라인 지원
- 모바일 최적화 UI

## 설치 및 사용 방법

### 1단계: 아이콘 생성

1. 컴퓨터 브라우저(Chrome, Safari 등)에서 `generate-icons.html` 파일을 엽니다
2. "192x192 다운로드" 버튼을 클릭하여 `icon-192.png`를 다운로드합니다
3. "512x512 다운로드" 버튼을 클릭하여 `icon-512.png`를 다운로드합니다
4. 다운로드한 두 아이콘 파일을 `HTML 뷰어` 폴더에 저장합니다

### 2단계: 웹 서버 실행

이 앱은 로컬 파일 시스템에서 직접 실행할 수 없으며, 웹 서버를 통해 실행해야 합니다.

#### 방법 1: Python 웹 서버 (권장)

Python이 설치되어 있다면:

```bash
# Python 3.x
cd "C:\Users\ehgus\Desktop\HTML 뷰어"
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000
```

#### 방법 2: Node.js 웹 서버

Node.js가 설치되어 있다면:

```bash
# http-server 설치 (한 번만)
npm install -g http-server

# 서버 실행
cd "C:\Users\ehgus\Desktop\HTML 뷰어"
http-server -p 8000
```

#### 방법 3: VS Code Live Server

VS Code를 사용하는 경우:
1. "Live Server" 확장 프로그램 설치
2. `index.html` 파일을 열고 우클릭
3. "Open with Live Server" 선택

### 3단계: 컴퓨터의 IP 주소 확인

웹 서버를 실행한 후, 컴퓨터의 로컬 IP 주소를 확인합니다.

**Windows:**
```bash
ipconfig
```
"무선 LAN 어댑터 Wi-Fi" 섹션에서 "IPv4 주소"를 찾습니다 (예: 192.168.0.10)

**Mac/Linux:**
```bash
ifconfig
```

### 4단계: iPhone에서 접속

1. iPhone이 컴퓨터와 **같은 WiFi 네트워크**에 연결되어 있는지 확인합니다
2. iPhone의 Safari 브라우저를 엽니다
3. 주소창에 입력합니다: `http://[컴퓨터IP주소]:8000`
   - 예시: `http://192.168.0.10:8000`
4. HTML 뷰어 앱이 열립니다

### 5단계: 홈 화면에 추가 (선택사항)

앱처럼 사용하려면:

1. Safari 하단의 **공유 버튼** (□↑) 을 탭합니다
2. 아래로 스크롤하여 **"홈 화면에 추가"** 를 선택합니다
3. 앱 이름을 확인하고 **"추가"** 를 탭합니다
4. 홈 화면에 "HTML 뷰어" 아이콘이 생성됩니다

이제 일반 앱처럼 홈 화면에서 바로 실행할 수 있습니다!

## 사용 방법

### 방법 1: 카카오톡/메신저에서 바로 열기 (추천!)

**앱을 홈 화면에 추가한 후에만 사용 가능합니다.**

1. 카카오톡, 이메일, 메신저 등에서 HTML 파일을 받습니다
2. 파일을 탭하여 다운로드합니다
3. 다운로드된 파일을 길게 누르거나 **"공유"** 버튼(□↑)을 탭합니다
4. 공유 메뉴에서 **"HTML 뷰어"** 앱을 선택합니다
5. 파일이 자동으로 열립니다!

**참고:** 공유 메뉴에 "HTML 뷰어"가 보이지 않으면:
- 앱을 홈 화면에 추가했는지 확인하세요
- 공유 메뉴를 아래로 스크롤해서 찾아보세요
- iPhone을 재시작해보세요

### 방법 2: 파일 앱에서 선택하여 열기

1. 앱을 실행합니다
2. 중앙의 **"HTML 파일 선택"** 박스를 탭합니다
3. iPhone의 파일 앱이 열리면 원하는 HTML 파일을 선택합니다
   - 여러 파일을 동시에 선택할 수 있습니다
4. 선택한 파일이 새 탭에서 열립니다

### 최근 파일 다시 열기

1. 앱 하단의 **"최근 파일"** 섹션에서 원하는 파일을 탭합니다
2. 해당 파일이 자동으로 열립니다

### 최근 파일 삭제

- 개별 삭제: 파일 옆의 **"삭제"** 버튼을 탭합니다
- 전체 삭제: 하단의 **"전체 삭제"** 버튼을 탭합니다

## iPhone으로 HTML 파일 전송하기

HTML 파일을 iPhone에서 열려면, 먼저 파일을 iPhone으로 전송해야 합니다.

### 방법 1: AirDrop (Mac 사용자)

1. Mac에서 HTML 파일을 선택합니다
2. 우클릭 > "공유" > "AirDrop"
3. iPhone을 선택합니다
4. iPhone에서 "파일" 앱에 저장합니다

### 방법 2: 이메일

1. 자신에게 HTML 파일을 첨부하여 이메일을 보냅니다
2. iPhone에서 이메일을 열고 첨부파일을 다운로드합니다
3. "파일" 앱에 저장합니다

### 방법 3: 클라우드 저장소

1. iCloud Drive, Google Drive, Dropbox 등에 HTML 파일을 업로드합니다
2. iPhone에서 해당 앱을 열고 파일을 다운로드합니다

### 방법 4: iTunes / Finder (유선 연결)

1. iPhone을 컴퓨터에 연결합니다
2. iTunes (Windows) 또는 Finder (Mac)에서 iPhone을 선택합니다
3. "파일 공유"로 이동합니다
4. "파일" 앱을 선택하고 HTML 파일을 추가합니다

## 문제 해결

### 앱이 로딩되지 않아요

- iPhone과 컴퓨터가 같은 WiFi에 연결되어 있는지 확인하세요
- IP 주소를 정확히 입력했는지 확인하세요
- 컴퓨터의 방화벽이 8000 포트를 차단하고 있지 않은지 확인하세요
- 웹 서버가 여전히 실행 중인지 확인하세요

### 파일을 선택했는데 열리지 않아요

- Safari가 팝업을 차단하고 있을 수 있습니다
- Safari 설정 > "팝업 차단" 을 해제하세요

### 3D 모델이 제대로 표시되지 않아요

- 인터넷 연결을 확인하세요 (three.js 라이브러리 로드에 필요)
- HTML 파일이 손상되지 않았는지 확인하세요

### 홈 화면에 추가가 안 돼요

- 반드시 **Safari 브라우저**를 사용해야 합니다 (Chrome은 불가)
- iOS 11.3 이상이 필요합니다

## 파일 구조

```
HTML 뷰어/
├── index.html              # 메인 앱 페이지

```

## 기술 스택

- HTML5 + CSS3 + JavaScript
- Progressive Web App (PWA)
- Service Worker (오프라인 지원)
- LocalStorage (최근 파일 저장)
- File API (파일 읽기)

## 브라우저 호환성

- iOS Safari 11.3 이상
- 홈 화면 추가: iOS Safari만 지원
- 파일 선택: iOS 13 이상 권장
