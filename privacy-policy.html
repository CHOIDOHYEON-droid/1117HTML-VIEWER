DLAS Viewer 개인정보 처리방침

최종 업데이트: 2025년 1월 19일

요약: DLAS Viewer는 사용자의 개인정보를 수집, 저장, 전송하지 않습니다. 모든 파일 처리는 사용자 기기 내에서만 이루어집니다.

1. 개요

DLAS Viewer("본 앱")는 HTML 파일을 열람하기 위한 간단한 뷰어 애플리케이션입니다. 본 앱은 사용자의 개인정보를 존중하며, 어떠한 개인정보도 수집하지 않습니다.

2. 수집하는 정보

본 앱은 어떠한 개인정보도 수집하지 않습니다.

수집하지 않는 정보:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 위치 정보
  • 기기 정보
  • 사용 통계 및 분석 데이터
  • 사진, 연락처 등 민감한 정보
  • 사용자가 열람하는 HTML 파일의 내용

3. 파일 처리 방식

본 앱에서 HTML 파일을 열람할 때:

  • 파일은 사용자 기기의 메모리에서만 처리됩니다
  • 파일 내용이 외부 서버로 전송되지 않습니다
  • 파일이 영구적으로 저장되지 않습니다
  • 뷰어를 닫으면 파일 내용이 메모리에서 제거됩니다

4. 로컬 저장소 사용

본 앱은 기술적으로 브라우저의 로컬 저장소(localStorage)를 사용할 수 있으나, 개인정보나 파일 내용을 저장하지 않습니다.

로컬 저장소는 앱의 상태 관리 목적으로만 사용되며, 외부 파일 열람 시 자동으로 초기화됩니다.

5. 제3자 서비스

본 앱은 다음과 같은 제3자 서비스를 사용하지 않습니다:

  • 광고 서비스
  • 분석 도구 (Google Analytics 등)
  • 크래시 리포팅 서비스
  • 소셜 미디어 연동
  • 클라우드 저장소

6. 인터넷 연결

본 앱은 HTML 파일을 열람하는 데 인터넷 연결이 필요하지 않습니다. 모든 처리는 오프라인으로 가능합니다.

7. 권한 요청

본 앱은 다음 권한만 요청합니다:

  • 파일 접근 권한: 사용자가 선택한 HTML 파일을 열람하기 위해 필요합니다

위치, 카메라, 연락처, 마이크 등의 민감한 권한은 요청하지 않습니다.

8. 데이터 보안

본 앱은 사용자 데이터를 수집하거나 저장하지 않으므로, 데이터 유출이나 무단 접근의 위험이 없습니다.

9. 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 포함한 모든 연령대의 사용자로부터 개인정보를 수집하지 않습니다.

10. 개인정보 처리방침 변경

본 개인정보 처리방침은 관련 법령 및 지침의 변경 또는 내부 정책 변경에 따라 변경될 수 있습니다. 개인정보 처리방침이 변경되는 경우, 앱 업데이트를 통해 공지하겠습니다.
